1. Download zip. 👆
1. Launch App!
1. Enter your Slack workspace name.
   ![sample_image](https://github.com/konoyono/okadash/blob/master/images/initialize.gif)

## Build and Run

1. clone this repo.
1. `npm install`
1. `npm start`
1. Enter your Slack workspace name.
